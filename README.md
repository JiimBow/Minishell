/*                                Description                                 */
/* ************************************************************************** */

The purpose of this project is to re-create a shell in a simplificate form.
The special characters with their fonctionnalities allowed are `|`, `<`, `>`, `<<`, `>>` and `$`

/* ************************************************************************** */
/*                                Instructions                                */
/* ************************************************************************** */

Do a: `make`
Then: `./minishell`
A prompt will appear like this: `minishell> `
Now you can launch commands like: `ls` `pwd` ...
Use the pipe or redirection like: `echo "Hello world" | cat -e > out`
